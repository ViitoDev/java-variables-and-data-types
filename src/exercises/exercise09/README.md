# Exercise 09 - Legal Age Check

## Objective

Show a message based on whether an age is at least 18.

## Concepts Practiced

- `int` and `String` variables
- Conditional statements
- Comparison operators
- Terminal output

## How It Works

The program checks the age value, stores the correct message in a variable, and prints that message.

## Example Output

```text
You are of legal age.
```
