# Exercise 06 - Price Category Classification

## Objective

Classify a product price into a simple price category.

## Concepts Practiced

- `double` variables
- `if`, `else if`, and `else`
- Comparison operators
- Multi-line terminal output

## How It Works

The program checks the price against category limits and prints both the price and its category.

## Example Output

```text
Price = 300.0
Category = Premium
```
