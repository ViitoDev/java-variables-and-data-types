# Exercise 10 - Trip Range Check

## Objective

Calculate vehicle range and check whether the current fuel level is enough for a planned trip.

## Concepts Practiced

- `double` variables
- Multiplication
- Conditional statements
- Comparison operators
- Terminal output

## How It Works

The program calculates the range for the current fuel level and for a full tank. It then compares the current range with the travel distance and prints whether refueling is needed.

## Example Output

```text
Current tank range: 250.0
Full tank range: 625.0
You can complete the trip without refueling.
```
