# Exercise 07 - Even or Odd Number Check

## Objective

Check whether an integer number is even or odd.

## Concepts Practiced

- `int` variables
- Modulo operator
- Conditional statements
- Terminal output

## How It Works

The program divides the number by two using the modulo operator. If the remainder is zero, it prints that the number is even; otherwise, it prints that the number is odd.

## Example Output

```text
The number is even
```
