# Exercise 04 - Celsius to Fahrenheit Conversion

## Objective

Convert a Celsius temperature to Fahrenheit.

## Concepts Practiced

- `int` and `double` variables
- Arithmetic expressions
- Temperature conversion
- Terminal output

## How It Works

The program stores a Celsius value, applies the Fahrenheit conversion formula, and prints the result.

## Example Output

```text
The Fahrenheit temperature is: 68.0
```
