# Exercise 03 - Grade Average Calculation

## Objective

Calculate and print the average of three grades.

## Concepts Practiced

- `double` variables
- Arithmetic operations
- Operator precedence
- Terminal output

## How It Works

The program adds three grade values, divides the total by three, and prints the calculated average.

## Example Output

```text
The average is: 5.0
```
