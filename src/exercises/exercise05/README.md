# Exercise 05 - Book Information and Category

## Objective

Store book information and show the category represented by a short code.

## Concepts Practiced

- `String`, `int`, and `double` variables
- Conditional statements
- String comparison with `equals`
- Terminal output

## How It Works

The program prints the stored book data and uses the category code to choose the matching category name.

## Example Output

```text
Title: 1984
Author: George Orwell
Pages: 416
Price: $13.5
The category is: Fiction
```
