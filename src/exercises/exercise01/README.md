# Exercise 01 - Basic Personal Data

## Objective

Store and print basic personal information using different Java data types.

## Concepts Practiced

- `String`, `int`, `double`, and `boolean` variables
- String concatenation
- Terminal output with `System.out.println`

## How It Works

The program creates variables for a name, age, height, and study status, then prints each value with a label.

## Example Output

```text
Name: Victor
Age: 20
Height: 1.76
Studying: true
```
