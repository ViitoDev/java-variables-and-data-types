# Exercise 08 - Currency Conversion

## Objective

Convert an amount in Brazilian reais to US dollars using a fixed exchange rate.

## Concepts Practiced

- `double` variables
- Division
- String concatenation
- Terminal output

## How It Works

The program divides the amount in reais by the dollar exchange rate and prints the converted value.

## Example Output

```text
R$50.0 is equal to US$9.578544061302683
```
