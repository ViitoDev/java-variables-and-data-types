# Exercise 02 - Explicit Type Casting

## Objective

Convert a decimal value to an integer using an explicit cast.

## Concepts Practiced

- `double` and `int` variables
- Explicit type casting
- Terminal output

## How It Works

The program stores a decimal number, casts it to an integer, and prints the resulting whole number.

## Example Output

```text
The integer value is: 10
```
